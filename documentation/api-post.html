Base URL: http://localhost:8000/api
GET /
Returns a list of all available API endpoints.

Sample cURL:

curl http://localhost:8000/api/
GET /posts
Fetch all blog posts.

Response:

[ { "id": 1, "title": "Example Post", "slug": "example-post", "content": "The first 100 characters...", "created_at": "2025-06-01T10:00:00Z", "updated_at": "2025-06-01T10:00:00Z" }, ... ]

Sample cURL:

curl http://localhost:8000/api/posts
GET /posts/{id}
Fetch a single post by ID.

URL Parameters:

id (integer): The post ID.

Sample cURL:

curl http://localhost:8000/api/posts/1
POST /posts/store
Create a new blog post.

Required JSON Body:

{ "title": "5 is the number of our fingers", "content": "Of course it is easier to count by five. We are humans after all.", "slug": "five-fingers", "is_published": true, "views": 0, "likes": 0 }

Sample cURL:

curl -X POST http://localhost:8000/api/posts/store \ -H "Content-Type: application/json" \ -d '{ "title": "5 is the number of our fingers", "content": "Of course it is easier to count by five. We are humans after all.", "slug": "five-fingers", "is_published": true, "views": 0, "likes": 0 }'
POST /posts/update/{id}
Update an existing post by ID.

URL Parameters:

id (integer): The post ID.

Sample cURL:

curl -X POST http://localhost:8000/api/posts/update/1 \ -H "Content-Type: application/json" \ -d '{ "title": "Updated Title", "content": "Updated content goes here.", "slug": "updated-title", "is_published": true, "views": 100, "likes": 10 }'
DELETE /posts/delete/{id}
Delete a post by ID.

URL Parameters:

id (integer): The post ID.

Sample cURL:

curl -X DELETE http://localhost:8000/api/posts/delete/1
